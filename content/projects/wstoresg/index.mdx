This is my first freelance project. The project is about ecommerce website that sells latex products.

### Technology stack

- Frontend enduser: Jquery, bootstrap.
- Frontend admin: Vuejs 3, Vue material, axios, vue-router.
- Backend: Spring boot, spring MVC.
- Database: Mysql 8.0.
- Server: VPS Centos 7.
- Deploy: Netlify, Apache Tomcat.
- Third-party: Cloudinary, VnPay.
