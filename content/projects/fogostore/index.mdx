### Technology stack

- Frontend enduser: Jquery, bootstrap.
- Frontend admin: Reactjs, antd, bootstrap-4, axios, redux, redux-saga, react-router-dom, create-react-app.
- Backend: Spring boot, spring MVC, Spring data JPA.
- Database: Mysql 8.0.
- Server: VPS Ubuntu 18.0.
- Deploy: Netlify, Apache Tomcat.
- Third-party: Cloudinary.
